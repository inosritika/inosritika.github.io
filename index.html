01. About Me

Hey, I'm Ritika 👋

I'm a Data Scientist at Oracle, but the part of the job I actually chase is the messy middle of applied AI: the gap between a model that works in a notebook and a system a real user can trust in production. I like taking a vague business ask and turning it into a retrieval path, an agent, or an activation flow that holds up under load.

My background is Mathematics and Computing at IIT Delhi, so I tend to reach for the fundamentals first. I care about why an index is O(n) instead of O(m·n), where latency is really hiding, and whether an agent is reasoning or just guessing. I would rather ship something a little smaller that I fully understand than a black box I can't debug at 2am.

And when I'm not shipping data platforms, I'm usually building something for fun: a fully local voice interviewer, a career-intelligence tool, a hackathon project put together overnight. I learn by making things end to end, which is why most of what I know lives on my GitHub rather than in a certificate.

What pulls me in

LLM systems that do real work: RAG and hybrid retrieval, agentic workflows, and real-time voice. The interesting problems live in latency, evals and trust, not just accuracy.

How I'm a little different

I optimize for systems I can reason about. Give me a fuzzy requirement and I will trace it down to complexity, failure modes and the one number that actually moves.

Off the clock

Weekend builds and hackathons, usually voice or agent projects. If an idea sounds fun at midnight, it tends to end up as a repo by the weekend.

03. Projects

Voice Interview Coach

A hands-free, fully local mock-interview coach. I built the whole real-time voice pipeline by hand (VAD → STT → LLM → TTS) and got response latency down to 2.75s with clause-level synthesis. It supports full-duplex barge-in using semantic turn detection, so it knows you're done talking from intent rather than silence, and the interviewer itself is an agent scored with LLM-as-judge evals.

Enterprise Work Intelligence (Codex Plugin)

An OraHacks 2026 project. A Codex plugin that pulls approved context from Outlook, Slack, Jira, Confluence, Zoom summaries and code repos into one prioritized work dashboard. Under the hood it runs a multi-agent workflow that spawns independent sub-agents for parallel investigation, planning and reply drafting, backed by a compact JSON memory so a fresh run starts with full context.
